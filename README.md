**KHELO** is a modern MERN stack application designed to bridge the gap between athletes and opportunities. It serves as a centralized platform where players from various sports (Football, Cricket, Tennis, etc.) can create professional profiles, track their statistics, manage achievements, and showcase their talent to scouts and teams.
